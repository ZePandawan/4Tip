Documentation 4Tip Discord Bot

Un bot Discord complet et personnalisable pour la gestion de serveurs et les interactions communautaires.

Commandes Disponibles

Commandes Administrateur

/warn

Description : Mettre un avertissement à un membre.

Paramètres :

  • membre : Mentionnez un membre (type : utilisateur)
  • raison : La raison du warn (type : chaîne de caractères, optionnel)

/add-streamer

Description : Ajouter un streamer à la liste des streamers à suivre.

Paramètres :

  • pseudo : Pseudo du streamer (type : chaîne de caractères, obligatoire)

/remove-streamer

Description : Supprimer un streamer de la liste des streamers à suivre.

Paramètres :

  • pseudo : Pseudo du streamer (type : chaîne de caractères, obligatoire)

/create-rules

Description : Créer les règles du serveur.

Paramètres :

  • role : Rôle à donner pour accepter les règles (type : rôle, obligatoire)

Commandes Utilisateur

/pokemon

Description : Attrapez des Pokémons.

/poke-inventory

Description : Voir votre inventaire de Pokémons.

Paramètres :

  • page : Page de l'inventaire à afficher (type : nombre, optionnel)

/list-streamer

Description : Afficher la liste des streamers à suivre.

/define-channel

Description : Définir les salons pour les messages du bot.

Paramètres :

  • type : Type de salon (choix entre `welcome`, `stream`, `leave`, `admin`) (type : chaîne de caractères, obligatoire)
  • channel : Salon à définir (type : salon, obligatoire)

/lol-stats

Description : Voir vos statistiques RANKED sur League of Legends.

Paramètres :

  • pseudo : Pseudo du joueur (type : chaîne de caractères, obligatoire)
  • tag : Tag du joueur (type : chaîne de caractères, obligatoire)
  • type : Choix entre `SOLO/DUO` ou `FLEX` (type : chaîne de caractères, obligatoire)

/lol-game-stats

Description : Voir vos 5 derniers matchs sur League of Legends.

Paramètres :

  • pseudo : Pseudo du joueur (type : chaîne de caractères, obligatoire)
  • tag : Tag du joueur (type : chaîne de caractères, obligatoire)
